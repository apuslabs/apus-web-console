## Prerequisites

### Software Requirements:
- [Git](https://git-scm.com/downloads)
- [Docker & Docker Compose](https://www.docker.com/get-started)

### Ethereum Wallet:
- [Wallet](../wallet) with at least `0.01 ETH` on Taiko for transactions.


### Hardware Specifications:
- CPU: `8/16` core
- RAM: `32GB` minimum
- Storage: `NVMe SSD` recommended
- Network: Stable connection with `9000` [port](./faq) open.
